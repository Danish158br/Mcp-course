MCP Kya Hota Hai? (What is MCP?)

MCP ka Full Form:

Model Context Protocol - Yeh ek special communication system hai jo AI models (jaise ChatGPT, Claude) aur different tools/services ke beech connection banata hai.

Simple Language Mein MCP:

Socho ke aap ek office mein boss hain, aur aapke paas bahut sare employees hain - koi Excel expert hai, koi database specialist hai, koi social media handler hai. Lekin sab different languages bolte hain!

MCP ek translator ki tarah kaam karta hai jo:

  • Aapki baat samajh kar har employee ko unki language mein task deta hai
  • Har employee ka kaam wapas laakar aapko organized form mein deta hai
  • Sab ko coordinate karta hai taake kaam smooth chale

MCP Ka Maksad:

1. AI ko Smart Banana

AI sirf text nahi, real tools use kar sake

2. Automation

Boring repetitive kaam khud-ba-khud ho jaaye

3. Integration

Different apps/services aasani se connect ho sakein

4. Efficiency

Ek jagah se multiple kaam control karna

Example Real Life Mein:

Agar aap YouTube channel chalate hain:

  • MCP trend analysis kar sakta hai (Google Trends se data)
  • Script likh sakta hai (AI writing tools se)
  • Thumbnail create kar sakta hai (Image generation tools se)
  • Video upload kar sakta hai (YouTube API se)
  • Social media pe promote kar sakta hai (Twitter/Instagram APIs se)

Sab kuch ek hi command se!

MCP ke Components (MCP Components)

MCP ek complete ecosystem hai jiski 4 main parts hain. Har part ka apna specific role hai:

1. MCP Server

Kya Karta Hai: Yeh actual tools aur services ko manage karta hai

Example: Database server, API gateway, file system

Think of it as: Restaurant ka kitchen - yahan actual cooking hoti hai

2. MCP Client

Kya Karta Hai: User interface aur AI model ko connect karta hai

Example: ChatGPT interface, Claude interface

Think of it as: Restaurant ka waiter - customer aur kitchen ke beech bridge

3. Protocol Layer

Kya Karta Hai: Communication rules aur standards define karta hai

Example: JSON messages, HTTP requests, WebSockets

Think of it as: Restaurant ka menu - sab ko pata hai kya order karna hai

4. Tools & Resources

Kya Karta Hai: Actual functionalities provide karta hai

Example: Database queries, file operations, API calls

Think of it as: Restaurant ke ingredients - actual kaam yahan se hota hai

MCP Architecture (MCP Ki Structure)

MCP ka architecture client-server model pe based hai, lekin yeh traditional architecture se kaafi different aur smart hai.

Architecture Ke Layers:

Layer 1: Presentation Layer (User Interface)

Components:

  • • Chat interfaces (ChatGPT, Claude)
  • • Web applications
  • • Mobile apps
  • • Command line tools

Responsibility:

User input lena aur results display karna

Layer 2: Application Layer (Business Logic)

Components:

  • • AI reasoning engine
  • • Task orchestration
  • • Context management
  • • Error handling

Responsibility:

Complex tasks ko simple steps mein break karna

Layer 3: Protocol Layer (Communication)

Components:

  • • JSON-RPC messaging
  • • WebSocket connections
  • • HTTP/HTTPS transport
  • • Message serialization

Responsibility:

Secure aur efficient communication ensure karna

Layer 4: Service Layer (Tool Integration)

Components:

  • • Database connectors
  • • API integrations
  • • File system access
  • • External service wrappers

Responsibility:

Real tools aur services ke saath interaction

Server Creation (MCP Server Banana)

Ab hum apna pehla MCP server banayenge! Yeh server different tools ko integrate kar sakta hai aur AI ke saath communicate kar sakta hai.

Basic Server Structure:

// src/server.ts

import { Server } from '@modelcontextprotocol/sdk/server/index.js'; import { StdioServerTransport } from '@modelcontextprotocol/sdk/server/stdio.js'; import { CallToolRequestSchema, ListToolsRequestSchema, } from '@modelcontextprotocol/sdk/types.js'; class MyMCPServer { private server: Server; constructor() { this.server = new Server( { name: 'my-first-mcp-server', version: '1.0.0', }, { capabilities: { tools: {}, }, } ); this.setupHandlers(); } private setupHandlers() { // Tools list karne ke liye this.server.setRequestHandler(ListToolsRequestSchema, async () => { return { tools: [ { name: 'calculator', description: 'Simple math calculations kar sakta hai', inputSchema: { type: 'object', properties: { operation: { type: 'string', enum: ['add', 'subtract', 'multiply', 'divide'] }, a: { type: 'number' }, b: { type: 'number' } }, required: ['operation', 'a', 'b'] } } ] }; }); // Tool execute karne ke liye this.server.setRequestHandler(CallToolRequestSchema, async (request) => { const { name, arguments: args } = request.params; if (name === 'calculator') { const { operation, a, b } = args as { operation: string; a: number; b: number; }; let result: number; switch (operation) { case 'add': result = a + b; break; case 'subtract': result = a - b; break; case 'multiply': result = a * b; break; case 'divide': result = a / b; break; default: throw new Error(`Unknown operation: ${operation}`); } return { content: [ { type: 'text', text: `Result: ${a} ${operation} ${b} = ${result}` } ] }; } throw new Error(`Unknown tool: ${name}`); }); } async run() { const transport = new StdioServerTransport(); await this.server.connect(transport); console.log('MCP Server started successfully! 🚀'); } } // Server start karna const server = new MyMCPServer(); server.run().catch(console.error);

Advanced Tools Add Karna:

File System Tool:

// File operations ke liye tool { name: 'file_operations', description: 'File read, write, aur list kar sakta hai', inputSchema: { type: 'object', properties: { action: { type: 'string', enum: ['read', 'write', 'list'] }, path: { type: 'string' }, content: { type: 'string' } // write ke liye }, required: ['action', 'path'] } } // Implementation case 'file_operations': const fs = require('fs').promises; const { action, path, content } = args; switch (action) { case 'read': const data = await fs.readFile(path, 'utf8'); return { content: [{ type: 'text', text: data }] }; case 'write': await fs.writeFile(path, content, 'utf8'); return { content: [{ type: 'text', text: 'File written successfully' }] }; case 'list': const files = await fs.readdir(path); return { content: [{ type: 'text', text: files.join('\n') }] }; }

Web API Tool:

// Weather API ke liye tool { name: 'weather', description: 'Current weather information get kar sakta hai', inputSchema: { type: 'object', properties: { city: { type: 'string' } }, required: ['city'] } } // Implementation case 'weather': const axios = require('axios'); const { city } = args; const apiKey = 'YOUR_API_KEY'; const url = `https://api.openweathermap.org/data/2.5/weather?q=${city}&appid=${apiKey}`; const response = await axios.get(url); const weather = response.data; return { content: [{ type: 'text', text: `${city} mein temperature ${weather.main.temp}K hai, weather ${weather.weather[0].description} hai` }] };

Client Implementation (Client Banana)

Server ke baad ab client banana hai! Client wo part hai jo user interface provide karta hai aur server ke saath communicate karta hai.

Basic Client Structure:

// src/client.ts

import { Client } from '@modelcontextprotocol/sdk/client/index.js'; import { StdioClientTransport } from '@modelcontextprotocol/sdk/client/stdio.js'; import { spawn } from 'child_process'; class MyMCPClient { private client: Client; private transport: StdioClientTransport; constructor() { this.client = new Client( { name: 'my-mcp-client', version: '1.0.0', }, { capabilities: {} } ); } async connect() { // Server process start karna const serverProcess = spawn('node', ['dist/server.js'], { stdio: ['pipe', 'pipe', 'inherit'] }); // Transport setup this.transport = new StdioClientTransport( serverProcess.stdout, serverProcess.stdin ); // Client connect karna await this.client.connect(this.transport); console.log('Client connected to server! ✅'); } async listAvailableTools() { try { const response = await this.client.request( { method: 'tools/list' }, { method: 'tools/list' } ); console.log('Available tools:'); response.tools.forEach(tool => { console.log(`- ${tool.name}: ${tool.description}`); }); return response.tools; } catch (error) { console.error('Error listing tools:', error); return []; } } async callTool(name: string, args: any) { try { const response = await this.client.request( { method: 'tools/call' }, { method: 'tools/call', params: { name: name, arguments: args } } ); return response; } catch (error) { console.error(`Error calling tool ${name}:`, error); throw error; } } async disconnect() { await this.client.close(); console.log('Client disconnected! 👋'); } } // Usage example async function main() { const client = new MyMCPClient(); try { await client.connect(); // Available tools dekhna await client.listAvailableTools(); // Calculator tool use karna const result = await client.callTool('calculator', { operation: 'add', a: 10, b: 5 }); console.log('Calculator result:', result); } catch (error) { console.error('Main error:', error); } finally { await client.disconnect(); } } main().catch(console.error);

JavaScript Client Logic:

// public/client.js class WebMCPClient { constructor() { this.connected = false; this.tools = []; } async connectToServer() { try { // WebSocket connection (requires WebSocket transport) this.ws = new WebSocket('ws://localhost:8080'); this.ws.onopen = () => { this.connected = true; document.getElementById('connection-status').innerHTML = '<span style="color: green;">✅ Connected</span>'; this.loadTools(); }; this.ws.onmessage = (event) => { const message = JSON.parse(event.data); this.handleMessage(message); }; this.ws.onerror = (error) => { console.error('WebSocket error:', error); document.getElementById('connection-status').innerHTML = '<span style="color: red;">❌ Connection failed</span>'; }; } catch (error) { console.error('Connection error:', error); } } async loadTools() { const message = { jsonrpc: '2.0', id: 1, method: 'tools/list' }; this.ws.send(JSON.stringify(message)); } handleMessage(message) { if (message.method === 'tools/list') { this.tools = message.result.tools; this.displayTools(); } } displayTools() { const container = document.getElementById('tools-container'); container.style.display = 'block'; this.tools.forEach(tool => { if (tool.name === 'calculator') { document.getElementById('calculator-tool').style.display = 'block'; } }); } async callTool(name, args) { const message = { jsonrpc: '2.0', id: Date.now(), method: 'tools/call', params: { name: name, arguments: args } }; return new Promise((resolve, reject) => { this.ws.send(JSON.stringify(message)); // Response handler (simplified) setTimeout(() => { resolve({ content: [{ text: 'Result received' }] }); }, 1000); }); } } const client = new WebMCPClient(); async function connectToServer() { await client.connectToServer(); } async function calculate() { const operation = document.getElementById('operation').value; const num1 = parseFloat(document.getElementById('num1').value); const num2 = parseFloat(document.getElementById('num2').value); if (isNaN(num1) || isNaN(num2)) { document.getElementById('calc-result').innerHTML = '<span style="color: red;">Please enter valid numbers</span>'; return; } try { const result = await client.callTool('calculator', { operation: operation, a: num1, b: num2 }); document.getElementById('calc-result').innerHTML = `<strong>Result:</strong> ${result.content[0].text}`; } catch (error) { document.getElementById('calc-result').innerHTML = `<span style="color: red;">Error: ${error.message}</span>`; } }

// YouTube Automation MCP Server class YouTubeAutomationServer { tools = [ { name: 'create_video_content', description: 'Complete video content pipeline - research se promotion tak', inputSchema: { type: 'object', properties: { topic: { type: 'string' }, target_audience: { type: 'string' }, video_length: { type: 'string' } } } } ]; async createVideoContent(args) { const { topic, target_audience, video_length } = args; // Step 1: Trend Research const trends = await this.googleTrendsAPI.getTrends(topic); const keywords = await this.keywordTool.getKeywords(topic); // Step 2: Script Generation const script = await this.aiWriter.generateScript({ topic, keywords, length: video_length, audience: target_audience }); // Step 3: Thumbnail Creation const thumbnail = await this.imageGenerator.createThumbnail({ title: script.title, style: 'viral', colors: ['red', 'yellow'] // High CTR colors }); // Step 4: SEO Optimization const seoData = await this.seoTool.optimize({ title: script.title, description: script.description, tags: keywords }); // Step 5: Video Upload (when ready) const uploadConfig = { title: seoData.optimizedTitle, description: seoData.optimizedDescription, tags: seoData.tags, thumbnail: thumbnail.url, category: 'Education', privacy: 'public' }; // Step 6: Social Media Content const socialPosts = await this.socialMediaTool.generatePosts({ platform: ['twitter', 'instagram', 'facebook'], content: script.summary, hashtags: keywords }); return { script: script, thumbnail: thumbnail, seo: seoData, uploadConfig: uploadConfig, socialMedia: socialPosts, analytics: { estimatedViews: trends.searchVolume * 0.02, potentialRevenue: trends.searchVolume * 0.02 * 0.001 // $1 per 1000 views } }; } } // Usage const result = await mcpClient.callTool('create_video_content', { topic: 'AI Tools for Students', target_audience: 'College students', video_length: '10 minutes' }); console.log('Video content ready!', result);

Automatic Retry Logic

async function retryOperation(operation, maxRetries = 3) { for (let attempt = 1; attempt <= maxRetries; attempt++) { try { return await operation(); } catch (error) { logger.warn(`Attempt ${attempt} failed`, { error: error.message }); if (attempt === maxRetries) { logger.error('All retry attempts failed'); throw error; } // Exponential backoff const delay = Math.pow(2, attempt) * 1000; await new Promise(resolve => setTimeout(resolve, delay)); } } } // Usage const result = await retryOperation(async () => { return await apiCall(); });

Circuit Breaker Pattern

class CircuitBreaker { constructor(threshold = 5, timeout = 60000) { this.failureThreshold = threshold; this.timeout = timeout; this.failureCount = 0; this.lastFailureTime = null; this.state = 'CLOSED'; // CLOSED, OPEN, HALF_OPEN } async execute(operation) { if (this.state === 'OPEN') { if (Date.now() - this.lastFailureTime > this.timeout) { this.state = 'HALF_OPEN'; } else { throw new Error('Circuit breaker is OPEN'); } } try { const result = await operation(); this.onSuccess(); return result; } catch (error) { this.onFailure(); throw error; } } onSuccess() { this.failureCount = 0; this.state = 'CLOSED'; } onFailure() { this.failureCount++; this.lastFailureTime = Date.now(); if (this.failureCount >= this.failureThreshold) { this.state = 'OPEN'; logger.warn('Circuit breaker opened due to failures'); } } }
